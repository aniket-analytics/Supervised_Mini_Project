This project focuses on predicting housing prices using the California Housing dataset through a complete machine learning workflow. It begins with Exploratory Data Analysis (EDA), where the dataset is inspected for missing values, basic statistics, and overall structure. Visualizations such as histograms and correlation heatmaps are used to understand the distribution of house prices and the relationships between variables. Important features like median income, average number of rooms, and house age are selected to build the prediction model. The dataset is then split into training and testing sets, and a Linear Regression model is trained to predict median house values. The model’s performance is evaluated using metrics such as Mean Squared Error (MSE) and R² score, and the relationship between actual and predicted prices is visualized using a scatter plot. Overall, the project demonstrates key machine learning concepts including data exploration, feature selection, model training, and regression model evaluation using Python and scikit-learn.
